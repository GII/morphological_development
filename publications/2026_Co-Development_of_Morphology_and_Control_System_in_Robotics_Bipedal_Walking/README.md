# Co-Development of Morphology and Control System in Robotics Bipedal Walking: Learning Outcomes

This folder contains the source code for the different types of the experiments, using as a reference those wiht α=0.5

In addition, it includes some videos showing different behaviors oftained, walking different distance values.

Experiments performed with the following setup:

- Ubuntu18
- Singularity 3.9.7
- CoppeliaSim 4.4.0.
- MultiNEAT implementation of the NEAT algorithm. Can be download from: https://anaconda.org/conda-forge/multineat
- The experiments were run on a computational cluster: https://cesga-docs.gitlab.io/
